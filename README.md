# AP2 Countdown & Stimmungsbarometer

> Weil man manchmal nicht nur wissen will, **wann** die AP2 kommt, sondern auch **wie sehr jeder gerade innerlich leidet**.  

## Features

- **Countdown** - die Uhr tickt. Kein Entkommen.  
- **Abstimmung** - zeig ehrlich, wie du dich fühlst.  
- **Ergebnisse** - Durchschnitt pro Fachrichtung plus Gesamtvotes. Sieh nach, ob dein Team besser oder schlechter dran ist als du.  
- **Stimmungs-Mapping** - aus Zahlen wird ein Label wie „Traurig“, „Neutral“ oder „Gut“. So erkennt man auf einen Blick, wer eigentlich am arsch ist.  

## Fachrichtungen

- FISI  
- FIAE  
- FIDV  
- FIDP  

## Benutzung

1. Abstimmen.  
2. Ergebnisse ansehen.  
3. Eventuell ein kleines Schulterzucken, wenn dein Score genau zwischen „Neutral“ und „Gut“ liegt.  

## Tech Stack

- **HTML/CSS/JS** - Reicht für diesen Kleinscheiß.  
- **Firebase Realtime Database** - für echtes Realtime-Leiden.
